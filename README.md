# Oasis Infobyte Internship - Task 1
## Retail Sales Data Analysis using Python

###  Project Overview
This project is part of the Oasis Infobyte Data Science Internship (Task 1). The objective is to analyze a retail sales dataset, clean the data, perform exploratory data analysis (EDA), generate visualizations, and provide business insights and recommendations.

###  Features
- Data Loading and Cleaning
- Missing Value & Duplicate Check
- Descriptive Statistics
- Monthly Revenue Trend Analysis
- Product Category Analysis
- Gender-wise Customer Analysis
- Heatmap Visualization
- Business Recommendations

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

###  Analysis Performed
- Dataset overview and preprocessing
- Mean, Median, Mode, and Standard Deviation
- Total Revenue Calculation
- Gender-wise Average Spending
- Monthly Revenue Trend
- Revenue by Product Category
- Gender Distribution Pie Chart
- Monthly Revenue Heatmap by Category

###  Visualizations
- Monthly Revenue Trend
- Product Category Revenue Bar Chart
- Gender Distribution Pie Chart
- Monthly Revenue Heatmap

###  Business Recommendations
- Increase stock for the best-performing product category.
- Offer discounts on low-performing categories.
- Focus marketing efforts during peak sales months.
- Reward loyal customers with special offers.
- Run flash sales during low-sales periods.

### Output Files
- `monthly_trend.png`
- `category_revenue.png`
- `gender_pie.png`
- `heatmap.png`
